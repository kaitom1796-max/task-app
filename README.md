# ToDo App

## 概要

FlaskとSQLiteを用いて制作しているタスク管理アプリです。

バックエンド開発やフロントエンドとの連携を学習することを目的として個人開発を行っています。  
タスクの追加・削除機能を実装しており、データベース操作や非同期通信を通して、Webアプリケーション開発の基礎を学習しています。

現在も機能追加・改善を継続しています。

---

## 使用技術

- Python
- Flask
- SQLite
- HTML
- JavaScript
- Git / GitHub

---

## 実装済み機能

### タスク追加機能
入力フォームからタスクを追加できます。

### タスク削除機能
完了ボタンを押すことでタスクを削除できます。

### データ永続化
SQLiteを使用し、アプリを再起動してもタスクが保持されるようにしています。

### 非同期通信
fetch APIを利用し、ページを再読み込みせずにタスクの追加・削除を反映できるようにしています。

---

## 学習したこと・工夫した点

- Flaskによるルーティング処理
- SQLiteを利用したデータベース操作
- fetch APIを利用したクライアント・サーバ間通信
- JavaScriptによるDOM操作
- SQLプレースホルダを利用した安全なDB操作
- フロントエンドとバックエンドの連携

また、機能ごとに処理を整理し、コードの可読性を意識して実装を行いました。
