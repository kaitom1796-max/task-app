# ToDo App

## 概要

FlaskとSQLiteを用いて開発したタスク管理Webアプリケーションです。
単純なToDo管理だけでなく、ユーザー認証機能を実装し、ユーザーごとにタスクを管理できるようにしています。さらに、実務を意識した高度な機能（REST API、自動テスト、CI/CDパイプライン）を組み込み、より実践的なアプリケーションへと発展させています。
単純なToDo管理だけでなく、ユーザー認証機能を実装し、ユーザーごとにタスクを管理できるようにしています。
バックエンド開発、データベース設計、フロントエンドとの連携を学習することを目的として個人開発を行いました。

## 起動方法

###ローカル環境
Bash

python app.py
ブラウザで以下へアクセスします。

Plaintext
[http://127.0.0.1:5000](http://127.0.0.1:5000)

Bash

Docker
Bash
