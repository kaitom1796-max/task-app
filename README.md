# ToDo App

## 概要

FlaskとSQLiteを用いて開発したタスク管理Webアプリケーションです。

単純なToDo管理だけでなく、ユーザー認証機能を実装し、ユーザーごとにタスクを管理できるようにしています。

バックエンド開発、データベース設計、フロントエンドとの連携を学習することを目的として個人開発を行いました。

## 起動方法

### ローカル環境

```bash
pip install -r requirements.txt
python app.py
```

ブラウザで以下へアクセスします。

```text
http://127.0.0.1:5000
```

### Docker

```bash
docker build -t todo-app .
docker run -p 5000:5000 todo-app
```

ブラウザで以下へアクセスします。

```text
http://localhost:5000
```
